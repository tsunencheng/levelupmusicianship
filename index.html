Built for the way classrooms actually run

Every game here started as a classroom-trialled activity, built around the same constraints any music teacher knows well: limited minutes, shared devices, mixed ability levels in one room, and the need for instant, visible feedback that feels like play, not a worksheet.

Games are levelled, not flat — so a class of mixed ability can all be playing the same game at the difficulty that fits them.

Looking for something screen-free? The Activities page has unplugged classroom games — the same kind of game-based thinking, no devices required.

Good to know

  • All games are free to use in any classroom.
  • No login required to play.
  • Works offline once a page has loaded once.
