Built for the way classrooms actually run

Every game here started as a classroom-trialled activity — not a theory exercise built for its own sake. They're designed around the same constraints any music teacher knows well: limited minutes, shared devices, mixed ability levels in one room, and the need for instant, visible feedback that doesn't feel like a worksheet.

Games are levelled, not flat — so a class of mixed ability can all be playing the same game and move up in levels at their pace.

No accounts, no data collection, no app installs. Every game runs as a single page that works on a shared classroom screen, a student laptop or a tablet.

Good to know

  • All games are free to use in any classroom.
  • No login required to play.
  • Works offline once a page has loaded once.
  • New games added as they're trialled and refined.
